# Multimodal RAG Chatbot

A Multimodal Retrieval-Augmented Generation (RAG) chatbot that answers questions from PDFs, images, and text documents using OCR, vector search, and Large Language Models (LLMs).

## Features
- PDF text extraction
- Image OCR using Tesseract
- Semantic search with embeddings
- Context-aware question answering
- Streamlit user interface

## Tech Stack
- Python
- Streamlit
- Sentence Transformers
- Tesseract OCR
- LangChain
- Gemini API

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
